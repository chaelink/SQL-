## 오라클 버전 별 차이 비교 정리

### 9i / 10g 버전
- CBO 중심 체계 확립

- 전통적인 NL Join 구조

- RBO → CBO 완전 전환 시기

- 실행계획 구조 단순

👉 보통 "11g 이전 버전"으로 묶여서 출제됨


### 11g 버전 이상

- NL Join Prefetch 도입

- Cardinality Feedback 도입

- Dynamic Sampling 강화

- 통계 기반 최적화 개선

#### 실행 계획
 - nl조인 시에 outer 테이블이 NESTED LOOPS 위에 존재하는 실행 계획은 없다
 - INNER 테이블이 NESTED LOOPS 위에 존재하는 경우는 있음 -> 테이블 prefetch 기능을 사용할 때
