## 챕터 2. SQL 분석 도구

### 1절 예상 실행계획

- 오라클 예상 실행계획 확인하기
  ```
  explain plan for
  select ~
  select * from table(dbms_xplan.display)
  ```  
- sql server 예상 실행계획 확인하기
  ```
  set showplan_text on
   ```
- 오라클 AutoTrace 확인하기
  ```
  set autotrace traceonly
  ```


<br>

- 오라클 예상 실행계획에서 확인할 수 있는 정보
  - 예상 cardinality
  - 예상 cost
  - 조건절 정보(Predicate Information)

<br>

- 오라클 AutoTrace
  - SQL을 실행하면서 실행 계획, 수행 통계를 함게 보여주는 기능 
- 오라클 AutoTrace에서 확인할 수 있는 정보
  - 예상 실행계획
  - 통계 정보
    - 실제 디스크에서 읽은 블록 수
    - 실제 기록한 redo 크기
